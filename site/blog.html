Введение в проект и подготовка технологического стека

Первое организационное собрание рабочей группы, в ходе которого были определены ключевые этапы реализации проекта. Проведен детальный анализ технического задания и сформирована дорожная карта разработки.

Основные выполненные задачи:

  • Проведен брифинг по проекту для всех участников команды
  • Проанализирована и утверждена архитектура проекта
  • Настроена система контроля версий (GitHub) с созданием основного репозитория

Освоение технологического стека:

TypeScript SCSS HTML5 JavaScript ES6+ Python 3.9

Технические решения: для обеспечения стабильности и масштабируемости проекта принято решение использовать современные веб-технологии с акцентом на кросс-браузерную совместимость и адаптивность интерфейса.

17 марта 2025

Отчёт об экскурсии по Центру карьеры Московского Политеха

Место проведения:

Центр карьеры Московского Политехнического Университета, Большая Семёновская улица, 40А, Москва

Цель экскурсии:

Ознакомление студентов с возможностями Центра карьеры, инструментами поиска работы, программами стажировок и взаимодействием с работодателями.

Сотрудники Центра карьеры рассказали о его основных функциях: помощь в трудоустройстве, организация ярмарок вакансий, подготовка к собеседованиям, сотрудничество с компаниями-партнёрами.

Продемонстрировали работу с порталом Центра карьеры, где размещаются актуальные стажировки и вакансии для студентов и выпускников.

Экскурсия позволила студентам:

  • Узнать о возможностях профессионального развития через сервисы Центра карьеры.
  • Были представлены ключевые сервисы: база вакансий, карьерные консультации, мероприятия с работодателями.
  • Разобраны типичные ошибки соискателей и даны рекомендации по самопрезентации.
  • Объявлены ближайшие мероприятия Центра карьеры (ярмарки вакансий, дни карьеры).
